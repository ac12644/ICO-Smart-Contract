### Installation

\_Below are instructions to get started

1. Clone the repo
   ```sh
     git clone https://github.com/ac12644/ICO-Token.git
   ```
2. Install packages
   ```sh
   npm install
   ```
3. Compile
   ```sh
   truffle compile
   ```
